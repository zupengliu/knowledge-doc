# Typora常用快捷键

### 文件

| 功能               | 热键（Windows / Linux） | 热键（macOS）       |
| :----------------- | :---------------------- | ------------------- |
| 新                 | Ctrl + N                | Command + N         |
| 新窗户             | Ctrl + Shift + N        | Command + Shift + N |
| 新标签             | *（不支持）*            | Command + T         |
| 打开               | Ctrl + O                | Command + O         |
| 快速打开           | Ctrl + P                | Command + Shift + O |
| 重新打开关闭的文件 | Ctrl + Shift + T        | Command + Shift + T |
| 保存               | Ctrl + S                | Command + S         |
| 另存为/重复        | Ctrl + Shift + S        | Command + Shift + S |
| 偏好设置           | Ctrl +，                | 命令+               |
| 关                 | Ctrl + W                | Command + W         |
